# ITC Clearface 폰트 파일

이 디렉토리에 ITC Clearface 폰트 파일을 추가해주세요.

## 필요한 파일

다음 파일들을 이 디렉토리에 추가하세요:

- `ITC-Clearface-Regular.woff2` (권장)
- `ITC-Clearface-Bold.woff2` (권장)
- `ITC-Clearface-Regular.woff` (대체)
- `ITC-Clearface-Bold.woff` (대체)
- `ITC-Clearface-Regular.ttf` (대체)
- `ITC-Clearface-Bold.ttf` (대체)

## 폰트 파일 형식 우선순위

1. `.woff2` (최신 브라우저, 가장 작은 파일 크기)
2. `.woff` (호환성 좋음)
3. `.ttf` (폴백)

최소한 `.woff2` 또는 `.woff` 형식의 파일이 필요합니다.

## 라이선스

ITC Clearface는 상업용 폰트입니다. 사용 전에 반드시 정식 라이선스를 구매하여 사용해야 합니다.
