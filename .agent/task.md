# Hero 섹션 별 아이콘 플로팅 애니메이션 적용

## 작업 개요

사용자의 요청에 따라 Hero 섹션의 오렌지 사각형 주변에 있는 별 아이콘(✦)에 둥둥 떠다니는 플로팅 애니메이션을 적용했습니다.

## 작업 상세

- [x] `components/Hero.module.scss`:
  - `@keyframes float-star` 정의: 위아래로 10px 움직이는 애니메이션
  - `.star1`: 3초 주기의 플로팅 애니메이션 적용
  - `.star2`: 4초 주기, 1초 지연의 플로팅 애니메이션 적용 (자연스러운 움직임 유도)

## 결과

Hero 섹션의 별 아이콘들이 부드럽게 움직이며 생동감을 더해줍니다.
